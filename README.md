# Documentation Example Code Validator

Validates example code snippets in documentation against a live environment.

## Free
```yaml
- uses: walshd1/documentation-example-code-validator@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```

## Paid (cost + 4.75%)
```yaml
- uses: walshd1/documentation-example-code-validator@v1
  with:
    service_token: ${{ secrets.ACTION_FACTORY_TOKEN }}
```
